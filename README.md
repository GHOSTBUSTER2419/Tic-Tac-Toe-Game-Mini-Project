## 🎮 Console Tic-Tac-Toe in C

A simple, lightweight, and interactive 2-player Tic-Tac-Toe game played directly in the terminal. Built collaboratively in C.

## ✨ Features
- **Classic Gameplay:** Standard 3x3 grid for two players ('X' and 'O').
- **Turn-based System:** Automatically alternates between players.
- **Input Validation:** Prevents players from choosing occupied squares or out-of-bounds coordinates.
- **Endgame Detection:** Accurately detects horizontal, vertical, and diagonal win conditions, as well as board draws.
